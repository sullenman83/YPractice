# YPractice
## Проектные работы в рамках обучения по программе "Продвинутая разработка на C# и .NET"

## В этой версии реализованы:
- REST API для работы с мероприятиями (CRUD). 
	- Операция получения всего спимка мероприятий
	- Операция получения мероприятия по id 
	- Операция создания нового мероприятия
	- Операция изменения мероприятия с заданным id
	- Операция удаления мероприятия по id
- Хранение мероприятий осуществляется в памяти
- Swagger для тестирования API
- Документация API


## Сборка осуществляется из директории репозитория командой 
dotnet build EventManagement\EventManagement.csproj 

## Запуск осуществляется из директории репозитория командой 
dotnet run --project EventManagement\EventManagement.csproj
